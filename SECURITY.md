# Security Policy

## Supported Versions

Currently, only the latest major release of CorePulse is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

** Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability within CorePulse, please send a private email to ssebuwufumoses1991@gmail.com. All security vulnerabilities will be promptly addressed and kept confidential until a patch is released.

**When reporting an issue, please include:**
* A description of the vulnerability (e.g., XSS, privilege escalation).
* Steps to reproduce the issue.
* The WordPress and CorePulse versions you were testing on.

We take the security of the WordPress community seriously and will work with you to verify, patch, and release a fix as quickly as possible.
